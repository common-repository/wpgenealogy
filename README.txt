WPGenealogy provides an amazing set of tools to help you create a complete family history website on WordPress.

== Description ==

__Easily build a family history site on your WordPress site__

The plugin for anyone wanting to build a family history website.  Manage and share family data with ease.

Thanks for checking us out.  We hope you like the plugin.  It's fairly new and quite powerful, but if you do happen to find a bug or have features you would like to see, please don't leave a bad review! Instead let us know by [posting feedback or help requests here](https://www.wpgenealogy.net/support/#/)  

This plugin includes -

= WPGenealogy Features  =
* Set up wizard to help you get started
* Create records for up to 200 family members (unlimited in Premium version)
* Multiple types of family chart ("family trees") showing ancestors and descendants
* Options to display vertical, horizontal, compact and text only charts
* Family member reports showing personal information and relatives
* Individual person time lines showing key personal events (plus can add additional events such as historical dates)
* Options to allow users to download family information via PDF
* Allow visitors to search for family members on multiple criteria
* Control access to your family information
* Enable logged in users to collaborate in your project with suggestions and editing rights
* Easy to configure family history hub landing page
* Shortcode enabling you to insert a descendants chart on any blog post or page

= WPGenealogy Premium Version =

* Import and export Gedcom format files
* Allow users to download Gedcom family files
* Add multiple separate families 
* Display an unlimited number of generations in a single tree
* Add photos to tree and family reports
* Style your family tree to suit your site - add colours (eg show male and female in different colour boxes)
* Add additional fields for recording different facts
* Enhanced shortcode add ancestor charts on any page or post in your blog

== Installation ==

###Manual Install From WordPress Dashboard

If your server is not connected to the Internet, then you can use this method-

1. Download the plugin by clicking on the blue 'Download' button above. A ZIP file will be downloaded.
2. Login to your site’s admin panel and navigate to Plugins -> Add New -> Upload.
3. Click 'Choose File', select the plugin file and click install.

###Install Using FTP

If you are unable to use any of the methods due to internet connectivity and file permission issues, then you can use this method-

1. Download the plugin by clicking on the blue 'Download' button above. A ZIP file will be downloaded.
2. Unzip the file.
3. Launch your favourite FTP client. Such as FileZilla, FireFTP, CyberDuck etc. If you are a more advanced user, then you can use SSH too.
4. Upload the folder to wp-content/plugins/
5. Log in to your WordPress dashboard.
6. Navigate to Plugins -> Installed
7. Activate the plugin.

== Frequently Asked Questions ==

= I found a bug, where can I report it?
  Let us know by [posting feedback or help requests here](https://www.wpgenealogy.net/support/#/)
= Where can I find documentation and support?
  Start [here](https://www.wpgenealogy.net/knowledge-base/#/)   
= Does WPGenealogy support WordPress multisite installation?=
  Yes.  
= Can I import/export Gedcom?=
  Yes. Although please note that Gedcom has a lot of variations and we cannot guarantee to import all data or every date format - check our support docs and keep back ups!
= I get an error 'Cannot redeclare...etc' when adding the Premium version.  How can I stop this?=
  Make sure the free version is disabled or removed to avoid this error message.
= Do you have an affiliate program?=
  Yes! You can earn 20% on sales of our Premium version by linking to our website or WordPress page - check out the Affiliation link in the plugin sidebar after installing.
= Can you build this of that feature - or my whole family history website?=
  Please contact us via our site [here](https://www.wpgenealogy.net/contact/) to discuss your bespoke requirements.
